# Personal Portfolio

This is a personal portfolio project showcasing my skills, projects, and professional links. The portfolio includes sections for my abilities, links to my GitHub repositories, LinkedIn profile, and published papers.

## Project Structure

- **src/**
  - **index.html**: Main HTML document for the portfolio.
  - **styles/**
    - **styles.css**: CSS styles for responsive design.
  - **scripts/**
    - **main.js**: JavaScript for interactive features.
- **assets/**
  - **fonts/**: Custom fonts used in the project.
  - **images/**: Images for the portfolio, including project screenshots and personal photos.
- **README.md**: Documentation for the project.
- **.gitignore**: Specifies files and directories to be ignored by Git.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd personal-portfolio
   ```
3. Open `src/index.html` in your web browser to view the portfolio.
