Projects

aoe2de_editor

aoe2de_editor was my first flutter project. It was born out of the frustration I experienced moding with the Genie editor. Instead of being a raw data editor like the Genie editor, aoe2de_editor is a more user-friendly editor showing the data in a way we as players are more familiar with instead of the low level details like the Genie editor does. The main purpose of this project is to help impleneting new civilization ideas in a much shorter time than it would take with the Genie editor.

Technologies used: Flutter, Dart

macchiato

macchiato is my latest project. It is a lprivacy first latex editor focused on making collaborative work like paper writing easier. It is designed to comply with the most strict privacy regulations and to be easy to use.

The macchiato project uses the power of a local LLM server to provide advance editing and writting features like literature suggestions, paper reviews, and more.

Technologies used: flutter, dart, python, C++, LLM

DH

This was a project I did as part of a lecture in university called Digital humanities. On this project I implemented a RAG system on a database with japanese news articles in Japanes and english. The main goal of this project was to create a system that could provide references to the articles it is using to as a source when answering questions.

Technologies used: Python, pandas, gemma (LLM), text encoders
